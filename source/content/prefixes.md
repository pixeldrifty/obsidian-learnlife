# Prefixes

| Part         | Meaning(s)              | Exemplars                       |
| ------------ | ----------------------- | ------------------------------- |
| un-          | not                     | unlock, unhappy                 |
| re-          | again                   | rebuild, return                 |
| pre-         | before                  | preview, prehistoric            |
| non-         | without, the absence of | nonprofit, nonsense             |
| de-          | remove, reverse         | defrost, debug                  |
| dis-         | not, opposite of        | disagree, disinfect, disappear  |
| uni- / mono- | one of something        |                                 |
| bi- / di-    | two of something        |                                 |
| anti-        | against                 |                                 |
| auto-        | self                    |                                 |
