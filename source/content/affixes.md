# Affixes

Pieces that attach to roots to change or add meaning. There are three types:

| Type   | Position        | Example         |
| ------ | --------------- | --------------- |
| Prefix | before the root | un-happy        |
| Suffix | after the root  | happiness       |
| Infix  | inside the root | rare in English |
