Commutative property: _u_ + _v_ = _v_ + _u_  
You can add numbers in any order and get the same sum.

![[Pasted image 20251027095010.png]]
Associative property:  
You can group the addends with parentheses and get the same sum.

![[Pasted image 20251027095047.png]]
Identity property:  
Adding zero does not change a number.

![[Pasted image 20251027095139.png]]
Distributive property:  
You can multiply a sum by multiplying each addend separately.

|                                        |                           |
| -------------------------------------- | ------------------------- |
| Commutative property of addition       | a + b = b + a             |
| Commutative property of multiplication | a x b = b x a             |
| Associative property of addition       | (a + b) + c = a + (b + c) |
| Associative property of multiplication | (a × b) × c = a × (b × c) |
| Distributive property                  | a(b + c)=a x b + a x c    |
