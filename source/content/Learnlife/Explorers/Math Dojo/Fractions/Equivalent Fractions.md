```
To simplify a fraction, divide the numerator and the denominator by the same number.
```
$$
16/8=8/4=4/2=2/1
$$
```
To compare two fractions, find the equivalent fractions with the same denominator.
```
$$
3/4 (</=/>?) 5/3
$$
```
To get the same denominator simply multiply one by the other:
```
$$
4*3 = 12
$$
```
Then to get the proper numerator multiply in a criss-cross pattern so for the left you would do:
```
$$
3*3=12
$$
```
And for the right you would do:
```
$$
5*4
$$
