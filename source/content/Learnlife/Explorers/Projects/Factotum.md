# Factotum
> Electron productivity suite with a tab-based interface for creative tools and development utilities.

## Overview
Factotum is a desktop productivity application built with Electron and React. It provides a unified workspace with multiple tabs covering creative tools, community forums, a development dependency graph, and settings.

## Tech Stack
- Electron 28
- React 19
- TypeScript
- Vite
- TailwindCSS
- framer-motion

## Key Features
- **Launcher tab** — creative tools hub
- **Forums tab** — community integration
- **Dev Graph tab** — visualize project/dependency relationships
- **Settings tab** — app configuration
